## 🚀 Comment Démarrer

### 1️⃣ Installation
```bash
cd PFE
npm install
```

### 2️⃣ Configuration
1. Créez un projet Firebase
2. Configurez `src/config/firebase.js`
3. (Optionnel) Ajoutez vos clés API dans `src/config/api.js`

### 3️⃣ Lancement
```bash
npm start
```
Puis scannez le QR code avec Expo Go !

📖 **Guide détaillé** : Voir [QUICK_START.md](QUICK_START.md)
